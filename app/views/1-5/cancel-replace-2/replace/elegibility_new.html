Do you need to replace this certificate?

If you still have the certificate, contact the importing country. They might accept an amended certified copy of the certificate, and you will not need to request a replacement.

If the consignment and the certificate have left the UK

You can request to replace the certificate.

If hand written changes are allowed

If the certificate does not have to be fully typed, you can amend the certificate and send it with the consignment.

If hand written changes are not allowed

You will need to request a replacement certificate. Make sure you include a written request from the importing country for the replacement along with your application.

! Warning APHA does not automatically replace certificates

APHA will not replace a certificate if:

  • the certificate has not been certified
  • the certificate is with the consignment, but it has not left the UK
  • you are allowed to amend the certificate by hand
