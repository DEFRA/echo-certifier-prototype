Request a replacement certificate

If the consignment and the certificate have left the UK

If hand written changes are allowed, amend and stamp the certified copy and include a letter on headed practice paper explaining why you have made the change. If you have any correspondence from the importing country via the importer, include that as well.

If hand written changes are not allowed, and you need to make a change to the certificate, you can request a replacement certificate.

If the consignment and the certificate has not left the UK

If you still have the certificate and hand written changes are allowed, contact the importer. They can check whether an amended certified copy of the certificate is allowed, and you may not need to request a replacement.

! Warning APHA does not automatically replace certificates

APHA will not replace a certificate if:

  • the certificate has not been certified
  • the certificate is with the consignment, but it has not left the UK
  • the certificate is lost in the post to certifier
  • you can make hand written changes
