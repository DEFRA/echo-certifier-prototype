Check if you need a replacement certificate

Check whether you need a replacement certificate for a consignment that has been certified and left the UK.

If you can make handwritten changes

If the importing country allows you to make handwritten changes to the certificate, then you don’t need a replacement.

If you can make handwritten changes:

  • make any changes you need to the certificate
  • stamp and initial those changes

Then, give it to the exporter along with:

  • a letter on headed paper explaining why you made the changes
  • any correspondence from the importing country

If handwritten changes are not allowed

If the importing country doesn’t allow you to make handwritten changes, you may need to ask for a replacement.

APHA won’t issue a replacement certificate if the original certificate:

  • has been lost or damaged
  • was not issued by the certifying officer

You don’t need a replacement certificate if the consignment has not yet left the UK.
